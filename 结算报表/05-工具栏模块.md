# 第五章 工具栏模块

## 5.1 页面头部动作

页面标题右侧提供 3 个操作按钮：

| 按钮 | 位置 | 说明 |
|------|------|------|
| 导出 CSV | 头部右侧 | 导出当前筛选条件下的完整结果集 |
| 导出 Excel | 头部右侧 | 导出当前筛选条件下的完整结果集 |
| 刷新数据 | 头部右侧 | 重新拉取报表数据并刷新 KPI 与表格 |

---

## 5.2 工具栏控件

| 控件 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 联赛筛选 | 下拉单选 | 全部联赛 | 仅影响当前筛选结果中的赛事 |
| 搜索框 | 文本输入 | 空 | 按赛事 ID 或球队名搜索 |
| 排序方式 | 下拉单选 | 按开赛时间排序 | 支持切换不同排序维度 |

---

## 5.3 搜索规则

| 规则项 | 说明 |
|--------|------|
| 搜索范围 | 赛事 ID、主队名、客队名 |
| 触发方式 | 输入后即时过滤，无需回车 |
| 清空搜索 | 恢复当前其他筛选条件下的全部结果 |
| 与其他筛选 | AND 关系 |

---

## 5.4 排序选项

| 选项 | 说明 |
|------|------|
| 按开赛时间排序 | 按赛事开赛时间排序，默认项 |
| 按结算时间排序 | 按赛事完成结算/取消的时间排序 |
| 按投注额排序 | 按赛事投注额从高到低排序 |
| 按盈亏排序 | 按平台盈亏从高到低排序 |

---

## 5.5 交互说明

| 场景 | 行为 |
|------|------|
| 切换联赛 | 立即刷新统计卡片和表格 |
| 切换排序 | 保持当前筛选条件，仅改变展示顺序 |
| 点击刷新数据 | 保留当前筛选、搜索、排序和分页状态 |

---

## 5.6 设计约束

一期原型未设计多选联赛、组合排序和保存常用筛选方案，本期不扩展这些能力。

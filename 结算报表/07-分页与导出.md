# 第七章 分页与导出

## 7.1 分页结构

分页器位于表格底部，左侧展示结果信息，右侧展示翻页控件。

| 区域 | 展示内容 |
|------|---------|
| 左侧信息 | `共 N 场赛事 | 显示 X-Y 条` |
| 右侧控件 | `上一页 | 当前页 / 总页数 | 下一页` |

---

## 7.2 分页规则

| 规则项 | 说明 |
|--------|------|
| 数据范围 | 表格分页只影响当前可见行，不影响顶部 KPI |
| 当前页合计 | 表格底部汇总行按当前页计算 |
| 切换分页 | 保留当前筛选、搜索和排序条件 |
| 首尾页按钮 | 首页禁用上一页，末页禁用下一页 |

---

## 7.3 导出按钮

页面头部提供两个导出按钮：

| 按钮 | 格式 |
|------|------|
| 导出 CSV | `.csv` |
| 导出 Excel | `.xlsx` |

---

## 7.4 导出规则

| 规则项 | 说明 |
|--------|------|
| 导出范围 | 当前筛选条件下的**全部结果集**，不只导出当前页 |
| 导出格式 | CSV / Excel 两种 |
| 文件命名 | `结算报表_YYYYMMDD_HHmmss.csv/xlsx` |
| 排序继承 | 导出结果应继承页面当前排序规则 |
| 空结果导出 | 无数据时按钮可点击，但返回“无可导出数据”提示 |

---

## 7.5 导出字段

| 序号 | 字段 |
|------|------|
| 1 | 赛事 ID |
| 2 | 联赛 |
| 3 | 主队 |
| 4 | 客队 |
| 5 | 比分 / 异常结果 |
| 6 | 结算来源 |
| 7 | 结算状态 |
| 8 | 盘口数 |
| 9 | 单关注单数 |
| 10 | 串关注单数 |
| 11 | 投注人数 |
| 12 | 投注额 |
| 13 | 派彩额 |
| 14 | 平台盈亏 |
| 15 | 盈利率 |
| 16 | 结算人员 |
| 17 | 开赛时间 |
| 18 | 结算时间 |

---

## 7.6 操作反馈

| 场景 | 反馈 |
|------|------|
| 点击导出 | 按钮进入 loading，防止重复点击 |
| 导出成功 | 提示“导出完成”并触发浏览器下载 |
| 导出失败 | 提示失败原因，如权限不足、数据生成失败 |

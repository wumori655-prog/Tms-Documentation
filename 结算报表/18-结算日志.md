# 第十八章 结算日志

> **关盘口径（2026-04-21 生效）**：关盘来源统一为数据源推送（唯一来源）；关盘 = 绝对终态，操盘页与结算详情页均不提供人工开盘入口。
>
> **核心规则引用**：日志字段结构、操作类型命名、操作人标注规则、日志保留策略 均以结算详情页第 21 章为核心规则。本章**完整复制** 核心规则 定义以便独立阅读；结算列表与结算详情页所有操作统一汇入同一日志库，按"来源页面"字段区分触发入口。

## 18.1 日志字段结构（与 settlement-detail 21.1 一致）
| **字段**  | **格式**                                                      | **说明**                                                                                           |
|-----------|---------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| 操作时间  | YYYY-MM-DD HH:mm:ss（UTC 时间戳记录，展示按用户本地时区转换） | 精确到秒                                                                                           |
| 操作人    | 结算人员姓名 / 结算主管姓名 / 「系统」                        | 见 18.3 操作人标注规则                                                                             |
| 操作人 IP | xxx.xxx.xxx.xxx                                               | 系统自动操作为空                                                                                   |
| 操作类型  | 见 18.2 操作类型清单                                          | 枚举写死，与 21.2 完全一致                                                                         |
| 操作对象  | 盘口 ID / 赛事 ID / 母单 ID / 注单 ID                         | 按操作类型决定                                                                                     |
| 操作内容  | 变更前 → 变更后                                               | 自由文本                                                                                           |
| 操作结果  | 成功 / 失败（含失败原因）                                     | —                                                                                                  |
| 关联信息  | 受影响的注单数 / 母单数 / 差额金额 / 累计次数等（若涉及）     | 按操作类型决定                                                                                     |
| 来源页面  | 结算详情页 / 结算列表-注单视图 / 结算列表-赛事视图 / 系统     | 用于跨模块审计（结算列表触发的日志来源为后两个值；结算详情页触发的为第一个值；系统自动为第四个值） |

## 18.2 日志记录范围（与 21.2 完全一致）
### 18.2.1 结算操作（两端可能触发）
| **操作类型**                            | **关联信息字段**                                                                                                                                                                               | **触发来源**                   |
|-----------------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------|
| 盘口结算（L3/L4 人工结算）              | 盘口 ID + 盘口名 + 结算结果 + 影响注单数 + 总派彩                                                                                                                                              | 结算详情页 / 结算列表-注单视图 |
| 盘口结算（L2 确认结算）                 | 盘口 ID + 盘口名 + 是否修改了系统预填结果 + 最终结算结果 + 影响注单数 + 总派彩                                                                                                                 | 结算详情页 / 结算列表-注单视图 |
| 盘口二次结算                            | 盘口 ID + 盘口名 + 原结算结果 + 新结算结果（如已重选）+ 影响注单数 + 已派彩金额 + 差额金额 + 关联串关母单数                                                                                    | 仅结算详情页                   |
| 数据源推送开盘信号触发的自动回滚（F10） | 盘口 ID + 盘口名 + 已结算→开盘 + 影响子注单数 + 影响串关母单数 + 原派彩总额 + 赛事级是否联动驳回 + 推送时间戳                                                                                  | 系统自动                       |
| 赛事驳回                                | 赛事 ID + 赛事名 + 原状态（已结算 / 已取消 / 异常态）→ 待结算 + 距最新结算时间剩余窗口（小时）                                                                                                 | 仅结算详情页                   |
| 赛事级确认结算                          | 赛事 ID + 赛事名 + 总盘口数 + 总投注额 + 总派彩                                                                                                                                                | 仅结算详情页                   |
| 赛事异常处理                            | 赛事 ID + 赛事名 + 异常原因（4 选 1：赛事取消 / 赛事延期 / 赛事腰斩 / 数据错误）+ 子方案（延期：暂缓 / 超时退款；腰斩：按比分 / 全额退款 / 暂缓）+ 涉及注单数 + 退款金额（如有）+ 赛事状态变化 | 仅结算详情页                   |

### 18.2.2 赛果录入操作（仅结算详情页）
| **操作类型** | **关联信息字段**                                                                |
|--------------|---------------------------------------------------------------------------------|
| 赛果录入     | 录入模式（数据源 / 人工 / 混合）+ 核心字段变更（比分 / 进球详情 / 角球 / 牌数） |

### 18.2.3 系统自动操作（操作人 = 「系统」）
| **操作类型**               | **关联信息字段**                                                                                           |
|----------------------------|------------------------------------------------------------------------------------------------------------|
| 串关母单自动结算           | 母单 ID + 子单系数列表 + 派彩金额 + 触发原因（首次结算 / 赛果变更）                                        |
| BT6 比分变动触发结算       | 盘口 ID + 选项 ID + 比分快照 + 判定结果 = 输                                                               |
| 赛果变化触发自动重算       | 盘口 ID + 盘口名 + 原结算结果 + 新结算结果 + 差额金额 + 影响单关数 + 影响母单数 + 该盘口累计赛果变更次数 N |
| 数据源盘口本地接管         | 盘口 ID + 盘口名 + 数据源未推送盘口终态时长 + 接管时间                                                     |
| 数据源盘口本地锁定生成     | 盘口 ID + 盘口名 + 触发场景（超时接管 / 赛事取消兜底）+ 锁定时间                                           |
| 数据源推送被本地锁定态拒收 | 盘口 ID + 盘口名 + IM 推送的比分/状态数据 + 本地锁定态比分/结果 + 拒收时间                                 |
| 赛事状态自动变更           | 赛事 ID + 变更前后状态 + 触发原因                                                                          |

### 18.2.4 管理级操作（仅结算列表，结算主管权限）
| **操作类型**           | **关联信息字段**                                         |
|------------------------|----------------------------------------------------------|
| 结算人员分配（单赛事） | 赛事 ID + 赛事名 + 阶段（赛前 / 滚球）+ 变更前后结算人员 |
| 批量分配结算人员       | 赛事 ID 列表 + 阶段 + 分配结算人员                       |

## 18.3 操作人标注规则（与 21.3 一致）
| **规则**             | **说明**                                                                                                               |
|----------------------|------------------------------------------------------------------------------------------------------------------------|
| 人工操作             | 记录结算人员 / 结算主管的姓名                                                                                          |
| 系统自动触发         | 记录「系统」                                                                                                           |
| 以下操作一律系统触发 | 赛果变化触发重算、串关母单自动结算、BT6 比分变动触发结算、数据源本地接管、数据源本地锁定、数据源拒收、赛事状态自动变更 |

## 18.4 日志展示入口
| **入口**          | **所属页面** | **展示范围**                                  |
|-------------------|--------------|-----------------------------------------------|
| 赛事视图-内嵌日志 | 结算列表     | 该赛事行底部展示最新 1 条，点击展开最近 20 条 |
| 右侧日志面板      | 结算详情页   | 当前赛事全部结算日志时间线                    |
| 盘口卡片内嵌日志  | 结算详情页   | 该盘口的结算日志（可展开最近 20 条）          |

## 18.5 日志保留（与 21.5 一致）
| **规则** | **说明**                                     |
|----------|----------------------------------------------|
| 保留期限 | 默认 365 天（系统级写死，修改需发版）        |
| 清理方式 | 清理前自动归档，归档保留时长由系统管理员配置 |
| 清理时机 | 每天凌晨自动执行                             |

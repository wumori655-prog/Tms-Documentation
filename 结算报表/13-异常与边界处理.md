# 第十三章 异常与边界处理

## 13.1 空结果

| 场景 | 处理 |
|------|------|
| 当前筛选无数据 | 表格区域展示空态，提示“当前条件下暂无结算报表数据” |
| 搜索无匹配 | 保留当前筛选，并展示“未找到匹配赛事” |

---

## 13.2 取消 / 异常赛事展示

| 场景 | 展示规则 |
|------|---------|
| 已取消赛事 | 状态标签为“已取消”，盈亏为 0，盈利率为 0% |
| 腰斩等异常结果 | 比分位置展示文本结果，如“腰斩” |
| 已取消但本地结算 | 来源标签仍按最终结算来源展示 |

---

## 13.3 跨日场景

| 场景 | 规则 |
|------|------|
| 开赛当天、结算次日 | 仍按开赛日期进入报表 |
| 结算时间晚于统计日 | 表格中如实显示次日结算时间，不做截断 |

---

## 13.4 大数字与长文本

| 场景 | 处理 |
|------|------|
| 金额很大 | 卡片允许缩写，表格保留完整数值 |
| 球队名过长 | 表格单元格优先保留赛事 ID 和操作列可读性，可截断展示 |
| 联赛标签过多 | 仅展示赛事所属联赛，不展示多级分类 |

---

## 13.5 导出异常

| 场景 | 处理 |
|------|------|
| 网络失败 | 提示“导出失败，请稍后重试” |
| 权限不足 | 提示“当前角色无导出权限” |
| 数据为空 | 提示“当前筛选条件下无可导出数据” |

---

## 13.6 查询口径边界

| 边界项 | 规则 |
|--------|------|
| 待结算赛事 | 不进入报表 |
| 异常处理中赛事 | 不进入报表 |
| 盘口二次结算 | 更新原记录，不新增一条重复赛事 |

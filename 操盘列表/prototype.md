# 交互原型
## 原型说明

### 功能覆盖

| 模块 | 状态 | 说明 |
|------|------|------|
| 侧边栏 | ✅ 完成 | 运动类型树、折叠功能 |
| 顶部栏 | ✅ 完成 | 统计卡片、一键锁盘 |
| 筛选区 | ✅ 完成 | 多条件筛选、快捷筛选 |
| 数据表格 | ✅ 完成 | 赛事列表、联赛分组 |
| 批量操作 | ✅ 完成 | 批量上架/下架、分配操盘手 |
| 弹窗交互 | ✅ 完成 | 上架确认、操盘日志 |
| 告警轮播 | ✅ 完成 | 多告警自动轮播展示 |

### 交互演示

- **点击「上架」按钮**：弹出上架确认弹窗
- **点击「详情」按钮**：弹出操盘日志弹窗
- **勾选赛事**：显示批量操作栏
- **点击联赛标题**：折叠/展开该联赛赛事
- **按 `?` 键**：显示快捷键提示

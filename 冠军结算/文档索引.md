# 冠军赛事结算详情页

冠军赛事（Outright）的结算操作页面，与单场常规赛事完全独立。处理夺冠预测、晋级预测、最佳球员、金靴奖等**长期盘口**的结算。

## 核心定位

| 维度 | 冠军赛事 | 对比常规赛事 |
|------|---------|------------|
| 数据源标识 | EventTypeId = 1 | EventTypeId = 2 |
| 时间跨度 | 整个赛季 / 锦标赛周期（数月）| 几十分钟到几小时 |
| 核心结果 | 单一赢家 / 多项排名 | 比分、分段数据等结构化赛果 |
| 结算触发 | 业务条件满足（官方公告）| 比赛结束 + 盘口关盘 |
| 赛事级"结算赛事"操作 | **没有**（系统自动推断）| 有（结算人员手动点击）|
| 统一赛果录入弹窗 | **没有**（按盘口独立录入）| 有（录入比分/进球等字段）|
| 24h 驳回窗口基准 | **盘口结算时间**（每盘口独立）| 赛事结算时间（赛事级）|

## 与常规赛事结算详情页的关系

冠军赛事与常规赛事是**两个完全独立的模块**：

- 入口：操盘列表 / 结算列表均有独立 Tab 区分「常规赛事」与「冠军赛事」
- 页面：`settlement-outright/` 与 `settlement-detail/` 互不依赖
- 业务规则：状态机、时间字段、录入流程均独立定义

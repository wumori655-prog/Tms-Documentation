# 第二章 页面结构

## 2.1 布局设计

冠军赛事结算详情页采用**三栏布局**，与常规赛事结算详情页结构类似但内容简化：

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶部导航栏（告警摘要 + 数据源状态 + 刷新）                                │
│ ※ 没有「赛果录入」按钮（冠军赛事无统一赛果录入弹窗）                       │
├──────────┬──────────────────────────────────┬───────────────────────┤
│          │ 赛事信息头（锦标赛名称 / 进度 / 取消按钮）│                       │
│ 左侧     │────────────────────────────────────│ 右侧                  │
│ 赛事切换栏 │ 盘口数据汇总（总数 / 已结算数 / 已取消数）│ 日志面板               │
│          │────────────────────────────────────│                       │
│ 待结算   │ 筛选工具栏（状态 / 搜索）            │ 搜索 + 筛选            │
│ 冠军赛事 │────────────────────────────────────│                       │
│ 快捷列表 │ 冠军盘口卡片 × N                    │ 日志时间线             │
│          │                                    │                       │
│ 状态图例 │                                    │                       │
└──────────┴──────────────────────────────────┴───────────────────────┘
```

## 2.2 三栏尺寸（与常规赛事一致）

| 区域 | 宽度 | 说明 |
|------|------|------|
| 左侧赛事切换栏 | 220px 固定 | 可折叠 |
| 中间操作区 | 自适应 | 核心工作区 |
| 右侧日志面板 | 320px 固定 | 可折叠 |

## 2.3 模块职责一览

| 模块 | 位置 | 职责 | 章节 |
|------|------|------|------|
| 左侧赛事切换栏 | 左侧 | 待结算冠军赛事快捷切换（仅显示 EventTypeId=1 的赛事）| 与常规赛事共享组件 |
| 赛事信息头 | 中间顶部 | 锦标赛名称、起止时间、进度、取消按钮 | [第 3 章](./03-赛事信息头) |
| 盘口数据汇总 | 中间 | 总盘口数 / 已结算数 / 已取消数 / 投注额 / 已派彩 | 结构同常规赛事 |
| 盘口筛选与 Tab | 中间 | 单 Tab（全部盘口）+ 状态筛选 | 无"本地 / 数据源" Tab 区分（冠军盘一律按业务规则处理） |
| 冠军盘口卡片 | 中间 | 选项列表 + 结算按钮 | [第 4 章](./04-冠军盘口卡片) |
| 右侧日志面板 | 右侧 | 结算日志时间线 | 结构同常规赛事 |

## 2.4 与常规赛事页面结构的差异

| 常规赛事模块 | 冠军赛事对应 | 差异 |
|------------|------------|------|
| 顶部「赛果录入」按钮 | **删除** | 冠军赛事无统一赛果录入弹窗 |
| 顶部「结算赛事」按钮 | **删除** | 冠军赛事无赛事级结算操作（系统自动推断）|
| 本地 / 数据源 Tab | **删除** | 冠军赛事不区分盘口来源 |
| 4 种渲染器卡片（MLT/SLT/Matrix/LongList）| 冠军盘口卡片（单一渲染器）| 冠军盘口通常是 LongList 或 SLT 形式，不存在让球线 |
| 赛事级赛事驳回按钮 | **删除** | 冠军赛事无赛事级驳回（改由盘口级二次结算处理）|
| 保留的按钮 | 取消锦标赛 | 整赛季取消的唯一赛事级操作 |

## 2.5 页面入口

| 入口 | 说明 |
|------|------|
| 操盘列表 · 冠军赛事 Tab → 点击赛事行 | 主入口 |
| 结算列表 · 冠军赛事 Tab → 点击赛事行 | 主入口 |
| 左侧赛事切换栏 → 点击其他冠军赛事 | 页内切换 |

# 第四章 工具栏模块

## 4.1 工具栏结构

工具栏位于主内容区顶部，包含视图切换和筛选控件。赛事视图和注单视图共享同一工具栏，注单视图额外展示注单类型 Tab。

## 4.2 控件定义

| **控件** | **类型**  | **默认值**                | **说明**                        |
| ------ | ------- | ---------------------- | ----------------------------- |
| 视图切换   | 按钮组     | 赛事视图（高亮）               | 切换时保留左侧筛选条件和滚动位置              |
| 联赛筛选   | 下拉单选    | 全部联赛                   | 动态加载（英超/西甲/德甲等），支持搜索定位        |
| 开赛时间   | 日期时间选择器 | 今天 00:00:00 ~ 23:59:59 | 精确到秒，两个输入框（开始/结束），按赛事开赛时间查询，格式 YYYY-MM-DD HH:MM:SS |
| 显示未完成  | 复选框     | 勾选                     | 勾选后同时展示开赛时间不在筛选范围内但仍有未结算盘口的赛事 |
| 时间快捷按钮 | 按钮组     | 今天（高亮）                 | 今天 / 昨天 / 本周，详见 4.4 节         |
| 结算人员   | 下拉单选    | 全部人员                   | 按已分配的结算人员筛选，支持搜索定位（输入姓名即时过滤），右侧提供清除按钮，详见 4.2.1 节 |
| 赛事搜索   | 文本输入框   | 空                      | 按赛事 ID 或盘口 ID 匹配              |

### 4.2.1 结算人员下拉的清除按钮（v1.1 补充）

开发实装时在「结算人员」下拉的当前选中值右侧加了「× 清除」按钮，文档原本未定义。本节补充入档：

| **控件** | **显示条件** | **位置** | **行为** |
| ------ | -------- | ------ | ------ |
| × 清除按钮 | 仅在当前已选中具体结算人员（非"全部人员"）时显示 | 下拉框内选中值的右侧 | 点击 → 恢复下拉值为「全部人员」（默认值），列表按"全部人员"重新筛选 |

> 「全部人员」状态下不显示 × 清除按钮（无可清除内容）。

## 4.3 搜索交互

| **规则项** | **说明**             |

## 4.4 时间快捷按钮

> ### ~~原定义（v1.1 已撤销，留底）~~
>
> ~~| **按钮** | **行为**           |~~
> ~~| ------ | ---------------- |~~
> ~~| 今天     | 开始 = 结束 = 当天     |~~
> ~~| 昨天     | 开始 = 结束 = 昨天     |~~
> ~~| 本周     | 开始 = 本周一，结束 = 今天 |~~
>
> ~~选中的快捷按钮高亮。手动修改日期后快捷按钮取消高亮。~~
>
> **撤销原因（v1.1，Mori）**：
>
> 1. **时分秒精度未定义**——「当天」「昨天」「本周一」等仅到日期层级，未明确具体时刻（00:00:00 / 当前时刻 / 23:59:59），会造成开发实装歧义、测试用例无法编写、边界 1 秒差异导致筛选结果不一致
> 2. **「本周」业务含义错误**——原定义「本周 = 本周一 ~ 今天」为「已发生的本周」，不符合用户预期的「完整一周赛程查看」语义；举例今日为周三，按原定义筛「本周」会漏掉周四 ~ 周日尚未开赛的赛事
> 3. **按钮与日期范围选择器的联动关系未定义**——原文未说明点击快捷按钮如何影响左侧「开赛时间」选择器的取值，仅模糊定义为「行为」

### 4.4.1 重新定义（v1.1）

时间快捷按钮的本质是**「开赛时间」选择器的快速填值器**：点击任一按钮后，工具栏左侧「开赛时间」控件（见 4.2 节）自动填入对应的开始/结束值，并按填入值执行筛选。

**填入值定义**（自然周口径：周一开始，周日结束）：

| **按钮** | **填入「开赛时间」选择器的值**                                                       |
| ------ | --------------------------------------------------------------------------- |
| 今天     | 开始 = 当日 00:00:00；结束 = 当日 23:59:59                                          |
| 昨天     | 开始 = 当日 -1 天 00:00:00；结束 = 当日 -1 天 23:59:59                                |
| 本周     | 开始 = 本周一 00:00:00；结束 = 本周日 23:59:59（完整一周，跨周/跨月/跨年按日期自然推算）                 |

**筛选执行**：按「开赛时间」选择器的最终值，匹配赛事「开赛时间」字段。

**联动与高亮规则**：

| **场景** | **行为** |
| ------ | ------ |
| 点击快捷按钮 | 「开赛时间」选择器值自动更新为对应预设值；该快捷按钮高亮 |
| 用户手动修改「开赛时间」选择器 | 三个快捷按钮全部取消高亮（说明当前为自定义范围） |
| 三个快捷按钮之间 | 互斥，同时仅有一个处于高亮态 |

### 4.4.2 统计与筛选联动（v1.2 新增）

时间快捷按钮按 4.4.1 节填入「开赛时间」选择器后，赛事列表按此时间范围执行筛选；**具体统计哪些状态的赛事，跟左侧筛选栏当前选中的「赛事结算状态」联动**。

**联动规则**：

| **关系项** | **说明** |
| ------- | ------ |
| 时间快捷按钮 × 左侧「赛事结算状态」 | **AND 关系**（同时满足） |
| 时间快捷按钮 × 左侧「赛事阶段」 | AND 关系 |
| 时间快捷按钮 × 左侧「运动类型」 | AND 关系 |

**数字徽章含义**：

按【当前左侧筛选条件】下，匹配【时间快捷按钮时间范围】的赛事总数。

**举例**：

| **左侧选择** | **时间按钮** | **数字含义** |
| -------- | -------- | -------- |
| 全部赛事 | 今天 | 今天开赛的全部赛事（待结算 + 已结算 + 异常 + 已取消） |
| 待结算赛事 | 今天 | 今天开赛的【待结算】赛事 |
| 异常赛事 | 本周 | 本周开赛的【异常】赛事（腰斩暂缓 + 延期暂缓） |
| 已结算赛事 | 昨天 | 昨天开赛、已结算、且在 24h 驳回窗口内的赛事 |

**数据范围硬约束**（继承 03 章 3.1 节）：

- 不包含赛前赛事（距开赛 > 30 分钟，在操盘列表管理）
- 即使时间范围包含更早时段，结果仍只显示其中已进入即将开赛或已开赛的赛事

**状态定义来源**：各结算状态的具体口径（含义、计数口径、徽章颜色、状态枚举）详见 [03 左侧筛选栏 3.2.1~3.2.5](https://github.com/wumori655-prog/Tms-Documentation/blob/main/%E7%BB%93%E7%AE%97%E7%AE%A1%E7%90%86/03-%E5%B7%A6%E4%BE%A7%E7%AD%9B%E9%80%89%E6%A0%8F.md)。

## 4.5 注单视图额外控件

切换到注单视图时，工具栏下方展示注单类型 Tab：

| **Tab** | **显示**    | **说明**   |
| ------- | --------- | -------- |
| 全部注单    | 全部注单(256) | 括号内为实时计数 |
| 单关      | 单关(180)   | 仅单关注单    |
| 串关      | 串关(76)    | 仅串关母单    |


Tab 右侧展示额外筛选：

| **控件**   | **选项**                              | **默认值**   | **说明**                                                                          |
| -------- | ----------------------------------- | --------- | ------------------------------------------------------------------------------- |
| 注单状态     | 全部状态 / 待结算 / 已结算 / 已取消              | 全部状态      | 对齐 settlement-detail 18.4 注单状态枚举（3 种：待结算 / 已结算 / 已取消），**已取消作为独立筛选值**，可单独查看已退款注单 |
| **投注阶段** | 全部 / 赛前 / 滚球 / 串关含赛前 / 串关含滚球 / 串关混合 | 全部        | 按注单**投注瞬间**的赛事阶段分类（赛前=投注时比赛未开赛 / 滚球=投注时比赛进行中）。串关母单按子单阶段组合分类                     |
| 全部结果     | （v1.1 待开发确认）                        | 全部结果      | 详见 4.5.3 节                                                                       |
| 无赛果原因    | 全部 / 本地模式 / 玩法不支持 / 数据源超时 / 赛事异常    | 全部        | —                                                                               |
| 注单搜索     | 文本输入                                | 按注单 ID 搜索 | —                                                                               |


### 4.5.1 投注阶段筛选

| **筛选值**   | **匹配注单**         | **说明**                     |
| --------- | ---------------- | -------------------------- |
| 全部        | 所有注单             | —                          |
| **赛前**    | 单关注单 + 投注时比赛未开赛  | 基准分 = 该时段最终比分（终场/半场）       |
| **滚球**    | 单关注单 + 投注时比赛进行中  | 基准分 = 下注后比分（终场比分 − 下注时比分）  |
| **串关含赛前** | 串关母单 + 所有子单为赛前   | 各子单按赛前公式独立计算系数             |
| **串关含滚球** | 串关母单 + 所有子单为滚球   | 各子单按滚球公式独立计算系数             |
| **串关混合**  | 串关母单 + 子单包含赛前和滚球 | 每子单按自身阶段独立计算，详见结算详情页 N7 附录 |

> **术语对齐**：对应操盘列表第 13 章 13.1 节 "赛前 / 滚球"定义；投注瞬间的赛事阶段（赛前/滚球）记入注单表。

### 4.5.2 注单视图工具栏定位

注单视图工具栏承担以下入口职责：

| **入口类别** | **说明** |
| -------- | ------ |
| 赛果处理     | 为待结算注单选择/确认赛果结算结果（L2 预填确认 / L3 / L4 人工选结果） |
| ~~批量复核~~ | ~~对一批已结算注单进行二次校核~~ |
| 异常派彩入口   | 进入异常注单（含赛事异常态下的注单）的派彩处理流程 |

> **「批量复核」撤销原因（v1.1，Mori）**：
>
> 1. **无产品定义**——开发实装的视图副标题文案中曾出现"批量复核"措辞（见 02.4.1 节），但该功能在产品文档（结算管理 / 结算详情）中均无对应章节定义
> 2. **一期未实装**——开发画面虽有该词，但实际未提供任何对应入口或操作
> 3. **与既有文档明文冲突**——本模块第 9 章 9.4 节明确写明「**注单视图无批量操作**」，"批量复核"与此约定直接矛盾
> 4. **职能重叠**——注单层面的二次校核已有完整机制：通过结算详情页的**赛事驳回**实现（参 12.0 节驳回时间窗规则，距最新结算时间 ≤ 24h 内允许）
>
> **保留删除线条目而非直接删除**：体现历史措辞的撤销痕迹，避免后续看到副标题文案中"批量复核"残影时产生困惑

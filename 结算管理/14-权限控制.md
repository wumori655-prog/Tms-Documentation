# 第十四章 权限控制

| 筛选与搜索                                          | ✔            | ✔            | ✔            | ✔            |
| 跳转结算详情                                        | ✔            | ✔            | ✔            | ✔            |
| **注单视图**                                        |              |              |              |              |
| 查看注单列表                                        | ✔            | ✔            | ✔            | ✔            |

## 14.2 敏感操作控制
| **规则项**          | **说明**                                                         |
|---------------------|------------------------------------------------------------------|
| 同人结算 + 二次结算 | 一期允许。同一结算人员可以对同一盘口执行结算及后续的盘口二次结算 |
| 四眼原则            | 一期不做（双人复核已确认暂缓）                                   |
| 权限不足提示        | 操作无权限时按钮不可见或禁用，不展示错误弹窗                     |

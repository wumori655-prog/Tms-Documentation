# 操盘系统 · 产品文档

欢迎使用操盘系统产品文档站。

## 📚 文档总览

本文档站包含以下 **8 大模块** 详细文档:

### [联赛管理](联赛管理/01-模块概述.md)
模块概述 + 交互原型与更新日志。联赛级配置(联赛状态、是否跟随数据源、串关支持等)。

### [操盘列表](操盘列表/01-产品概述.md)
21 篇文档,涵盖产品概述、页面结构、侧边栏、筛选、批量操作、数据表格、权限控制、状态流转、排序规则、流程图等。

### [操盘详情](操盘详情/01-模块概述.md)
16 篇文档,涵盖盘口工具栏、盘口卡片、赔率编辑、控制层级体系、状态流转、数据源开关、监控面板、操作流程等。

### [结算管理](结算管理/01-产品概述.md)
19 篇文档,涵盖左侧筛选栏、工具栏、赛事/注单视图、串关展开、弹窗交互、批量操作、字段定义、状态流转、操作流程、权限、实时数据、异常处理、配置汇总、结算日志、流程图等。

### [结算详情](结算详情/01-产品概述.md)
11+ 篇文档,涵盖盘口筛选与 Tab、盘口结算卡片、弹窗交互、右侧日志面板、赛果数据录入规则、串关结算规则、赛事异常处理、权限控制、结算日志、结算场景手册、赛果录入与玩法结算映射表等。

### [结算报表](结算报表/05-工具栏模块.md)
6 篇文档,涵盖工具栏、结算明细表、分页与导出、异常处理、配置项汇总等。

### [冠军结算](冠军结算/01-产品概述.md)
9 篇文档,涵盖产品概述、页面结构、冠军盘口卡片、盘口结算规则、取消规则、权限控制、时间字段定义、结算日志等。

### [风控管理](风控管理/01-模块概述.md)
7 篇文档,涵盖赛事限额、用户限额、玩法限额与分组、串关限额、风险敞口计算、告警阈值等。

## 🔍 使用提示

- **左侧导航**:浏览所有模块和章节
- **顶部搜索框**:全文搜索关键字(也可以直接输入 `/` 快速聚焦搜索框)
- **右侧目录**:当前文档的章节锚点,点击跳转
- **顶部 Tabs**:快速在 8 大模块间切换

# 交互原型

## 原型说明

### 功能覆盖

| 模块 | 状态 | 说明 |
|------|------|------|
| 联赛列表 | ✅ 完成 | 所有联赛的列表展示与筛选，等级(默认+1~10)共11档 |
| 联赛配置 | ✅ 完成 | 联赛等级(默认+等级1-10共11档)、数据源跟随、串关、主力线规则 |
| 玩法配置 | ✅ 完成 | 足球208种FT玩法按6组展示（让球3种、大小24种、角球34种、进球100种、半场14种、特殊33种；可操盘82种+不可操盘126种），支持单独启用/禁用 |
| 操盘手分配 | ✅ 完成 | 穿梭框按6组快捷选择，赛前/滚球分阶段分配 |

### v0.3 更新内容（2026-02-12）

- 联赛等级补充「默认」档，从等级1-10（10级）调整为默认+等级1-10（11档），与风控管理联赛分组对齐
- 足球玩法列表从17种可操盘玩法扩展为208种FT完整6组分组（让球、大小、角球、进球、半场、特殊，含全部比赛级别玩法；可操盘82种+不可操盘126种）
- 删除BT11（冠军）：BT11为锦标赛级别玩法，不属于比赛级别6组分组
- 原B组从16种恢复为15种（已按新6组架构重新分配）

### v0.2 更新内容（2026-02-11）

- 联赛等级从 S/A/B/C（4级）调整为 等级1-10（10级），与风控管理联赛分组对齐
- 足球玩法列表维持17种可操盘玩法（BT48/BT179/BT180已纳入可操盘，BT18/BT22/BT23/BT273不纳入可操盘范围）
- 玩法列表增加6组分组标识（与风控管理玩法限额一致）
- 操盘手穿梭框快捷按钮从"核心/特殊"改为6组分组选择
- 配置弹窗等级选择从radio改为select下拉

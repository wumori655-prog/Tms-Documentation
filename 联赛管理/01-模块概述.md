## 1.1 模块定位

联赛管理是操盘系统的配置中心，用于管理联赛级别的操盘参数。配置变更会影响该联赛下所有赛事的行为。

## 1.2 核心配置项

| 配置项 | 说明 | 影响范围 |
|--------|------|----------|
| 联赛状态 | 开启/暂停/关盘 | 决定赛事是否可上架 |
| 联赛等级 | 默认+等级1-10共11档 | 关联限额配置（由风控管理统一配置）；未分配等级的联赛使用「默认」档配置 |
| 跟随数据源盘口状态 | 默认值为「是」（联赛管理配置） | 控制IM暂停等状态是否影响C端展示；IM暂停不改变本地状态，仅影响C端显示暂停投注 |
| 默认操盘手 | 默认值为空（联赛管理配置） | 赛前/滚球分别配置，批量上架时的默认分配 |
| 串关支持 | 是/否 | 该联赛赛事是否支持串关 |
| 玩法启用列表 | 默认值为全部启用（联赛管理配置） | 一期足球208种FT玩法（可操盘82种+不可操盘126种），按6组管理（让球3种、大小24种、角球34种、进球100种、半场14种、特殊33种），分组定义以风控管理第四章为SSOT |
| 主力线显示规则 | 默认值为「仅显示主力」（联赛管理配置） | 控制副线是否默认展示 |

### 1.2.1 UI显示规范

| UI元素 | 显示文案 | 说明 |
|--------|---------|------|
| 联赛等级下拉-默认档 | "默认（未分配等级）" | 未分配等级的联赛使用此档 |
| 联赛等级下拉-等级1~10 | "等级1" ~ "等级10" | 数字越小等级越高 |
| 筛选器-默认档选项 | "默认（未分配）" | 筛选器中的默认档选项 |

## 1.3 配置生效规则

::: tip 重要原则
联赛配置变更仅影响**未上架赛事**，已在操盘列表中的赛事不受影响。
:::

| 配置变更 | 对已上架赛事 | 对待上架赛事 |
|----------|--------------|--------------|
| 联赛禁用 | 不影响 | 从列表中移除 |
| 联赛等级变更 | 不影响（保持原等级限额） | 新上架时使用新等级限额 |
| 跟随配置变更 | 立即生效（例外，见操盘列表第16章） | 按新配置生效 |
| 默认操盘手变更 | 不影响 | 新上架时使用新配置 |
